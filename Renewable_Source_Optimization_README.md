# 📊 Renewable Energy Production & Capacity Analysis

This project was developed as part of my **final year academic project**, focusing on analyzing global renewable energy data across multiple sources such as solar, wind, hydro, biomass, and geothermal.

The objective of this project is to identify **energy production trends, installed capacity, investment patterns, and country-wise performance** using data analytics techniques and interactive visualization.

---

## 🚀 Project Overview

- Analyzed renewable energy data to uncover trends and insights  
- Built an **interactive Power BI dashboard** for visualization  
- Compared performance across different energy sources and countries  
- Performed **time-series analysis** to study growth patterns over the years  
- Designed KPIs for better understanding of energy production and sustainability  

---

## 📊 Dashboard Features

- Energy source-wise analysis (Solar, Wind, Hydro, Biomass, Geothermal)  
- Country-wise renewable energy production ranking  
- Capacity utilization comparison across countries  
- Investment and installed capacity insights  
- Time-based trend analysis  
- Interactive filters for year and energy type  

---

## 🛠️ Tools & Technologies

- Power BI  
- Microsoft Excel

This project uses a **publicly available dataset from Kaggle** containing information on renewable energy production, installed capacity, and investments.
