# Bank Analytics Project

## Project Overview
This project analyzes banking and loan data to uncover trends, loan distribution, and customer behavior. It helps support data-driven decision-making in banking operations.

## Features
- Total loans and loan amount analysis
- Average loan amount and interest rate
- Year-wise loan trends
- Customer segmentation
- Interactive dashboards for visualization

## Tools Used
- SQL for querying data
- Excel for data cleaning
- Power BI / Tableau for dashboards

## Dataset
Key fields in the dataset:
- `loan_amnt`: Loan amount
- `int_rate`: Interest rate
- `issue_d`: Loan issue date
- Customer details: age, income, location, etc.

## How to Run
1. Import the dataset into your preferred tool (SQL, Excel, Python).  
2. Execute queries or scripts to analyze data.  
3. Load processed data into Power BI / Tableau to create dashboards.  
4. Explore dashboards for insights.

## Key Insights
- Loan distribution and trend analysis
- Customer segmentation and behavior insights
- Support for data-driven banking strategies
